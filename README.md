# OpenSharpUE2IvanovKirillResearchProject

Я стал программистом потому что когда увидел у одноклассника впервые эту игру это был целый мир, тогда не было онлайн игр вообще и это была та игра где был огромный мир и игроки были живые, это казалось волшебством и поэтмоу я хотел научиться делать такие миры, но увы сейчас невозможно вернуть то время и даже запустить старый клиент на современных платформах и ChatGPT на основе Gildor-проекта помог мне снова открыть дверь в этот мир, увидеть его внутри, то как великие творцы прошлого создавали шедевры и стать хотябы косвенно частью разработки старого шедевра. Возможно, энтузиасты или старые авторы смогут сделать хорошую РПГ, как это например было с "New Vegas".

Этот проект — портфолио, открытое исследование данных Unreal Engine 2 на C#.

Поддерживаемые типы файлов: `.unr`, `.ukx`, `.usx`, `.utx`.

Отдельная благодарность авторам Blender-аддона для импорта PSK/PSA: `Darknet`, `flufy3d`, `camg188`, `befzz`, `floxay`. Без их работы у меня бы не получилось довести анимации до корректного состояния.

Как использовать:
собрать solution, укажите путь к локальному UE2-клиенту Lineage и запускать viewer для проверки пакетов, карт и сценовых объектов.

English:

I became a programmer because, when I first saw this game at a classmate’s place, it felt like an entire world. Back then, online games barely existed at all, and this was a game with a huge world where the players were real people. It felt magical, and that is why I wanted to learn how to create worlds like that.

Sadly, it is now impossible to bring that time back, and even running the old client on modern platforms is difficult. But with the help of ChatGPT, based on Gildor’s project, I was able to open the door to that world again, see it from the inside, understand how the great creators of the past built their masterpieces, and become, at least indirectly, part of the development of an old classic.

Perhaps enthusiasts or the original authors will someday be able to create a great RPG again, similar to what happened with *New Vegas*.

This project is a portfolio piece and an open research project focused on Unreal Engine 2 data in C#.

Supported file types: `.unr`, `.ukx`, `.usx`, `.utx`.

Special thanks to the authors of the Blender add-on for importing PSK/PSA files: `Darknet`, `flufy3d`, `camg188`, `befzz`, `floxay`. Without their work, I would not have been able to bring the animations to a correct state.

How to use:

Build the solution, specify the path to a local UE2 Lineage client, and run the viewer to inspect packages, maps, and scene objects.
